# 音效檔（SFX）

把音效檔依「事件名稱」放在此目錄即可生效，**程式會自動依檔名載入，不必改任何程式碼**。
缺檔時該音效靜音、不會報錯（可逐步補齊）。

## 命名慣例

檔名 = 事件名，副檔名 `.mp3`，例如 `swing.mp3`。

| 檔名             | 觸發時機                                   | 目前狀態 |
| ---------------- | ------------------------------------------ | -------- |
| `footstep.mp3`   | 走路腳步（按一下一聲、按住連續）           | ✅ 已放  |
| `swing.mp3`      | 近戰／衝刺類出手（普攻揮砍）               | 待補     |
| `cast.mp3`       | 施法／遠程／增益類出手                     | 待補     |
| `dash.mp3`       | 衝刺技                                      | 待補     |
| `blink.mp3`      | 瞬移／閃現技                                | 待補     |
| `ultimate.mp3`   | 大招施放                                    | 待補     |
| `hit.mp3`        | 攻擊命中／撞擊                              | 待補     |
| `hurt.mp3`       | 角色受傷（HP 下降）                         | 待補     |
| `death.mp3`      | 角色死亡                                    | 待補     |
| `buff.mp3`       | 增益／護盾（如需與 cast 區分時）           | 待補     |

## 每角色專屬音效（可選，會覆寫泛型）

以該動作的 **vfx id** 命名即可覆寫上表的泛型音效；缺檔時自動回退泛型名。
範例：

- `warrior_grapple.mp3` → 戰士「鉤索」專屬出手音（缺檔則用 `swing.mp3`）
- `mage_fireball.mp3`    → 法師火球專屬施法音（缺檔則用 `cast.mp3`）
- `fighter_ultimate.mp3` → 格鬥家大招專屬音（缺檔則用 `ultimate.mp3`）

vfx id 對應請見 `src/game/characters.js` 各角色 `basic/skill1/skill2/ultimate` 的 `vfx` 欄位。

## 檔案格式

- 建議 MP3（相容性最佳）；瀏覽器支援的 WAV／OGG／M4A 亦可。
- 短音效請保持精簡（出手／命中通常 < 1 秒），`footstep.mp3` 可為連續多步的素材（程式會自動切片輪播單步）。

## 行為說明

- **空間化**：音效依與本地玩家的距離自動衰減音量，並做左右聲道定位（遠處角色較小聲）。
- **觸發分工**：出手／施法／大招／受傷／腳步由渲染端依角色狀態觸發；命中／死亡由特效事件觸發（不會重複發聲）。
- **音樂**：背景音樂在 `../music/`，與此處音效系統各自獨立。
