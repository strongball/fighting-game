# 音效檔（SFX）

把音效檔依「事件名稱」放在此目錄即可生效，**程式會自動依檔名載入，不必改任何程式碼**。
缺檔時該音效靜音、不會報錯（可逐步補齊）。

> 目前的 `.wav` 由 **`tools/gen_sfx.py`** 程序化合成（擬真厚重、每角色一套簽名音色）。
> 要換成正式素材，放上同名的 `.mp3`（會優先採用）即可，合成的 `.wav` 可刪可留。
>
> **想微調某個音效**：改 `tools/gen_sfx.py` 裡該名稱的 `reg(...)` 配方，再跑
> `python3 tools/gen_sfx.py <名稱…>`（不帶參數＝全部重生）。合成是確定性的（依名 seed），可重跑。

## 命名慣例

檔名 = 事件名，副檔名 `.mp3`（正式，優先）或 `.wav`（佔位）；同名兩者都在時以 `.mp3` 為主。例如 `swing.mp3`。

| 檔名（泛型） | 觸發時機                             | 目前狀態      |
| ------------ | ------------------------------------ | ------------- |
| `footstep1` `footstep2` `footstep3` | 走路腳步（按一下一聲、按住連續；每步隨機選一） | 🔊 合成     |
| `swing`      | 近戰／衝刺類出手（普攻揮砍）             | 🔊 合成     |
| `cast`       | 施法／遠程／增益類出手                 | 🔊 合成     |
| `dash`       | 衝刺技                                 | 🔊 合成     |
| `blink`      | 瞬移／閃現技                           | 🔊 合成     |
| `ultimate`   | 大招施放                                 | 🔊 合成     |
| `hit`        | 攻擊命中／撞擊                           | 🔊 合成     |
| `hurt`       | 角色受傷（HP 下降）                      | 🔊 合成     |
| `death`      | 角色死亡                                 | 🔊 合成     |
| `buff`       | 增益／護盾（如需與 cast 區分時）        | 🔊 合成     |

## 每角色每招專屬音效（覆寫泛型）

出手音以**技能槽**為鍵查找（renderer.js），讓**每個角色的四個技能各有獨立音色**：

1. **`<charId>_<slot>`**（主）— `slot` ∈ `basic / skill1 / skill2 / ultimate`。
   例 `warrior_basic`（橫掃）、`tank_skill2`（巨力踏陣）、`healer_ultimate`（生命匯流）。
   `<charId>` = 角色資料夾 slug（含連字號，如 `magic-swordsman_basic`、`star-orbit_ultimate`）。
2. **泛型**（`swing / cast / dash / blink / ultimate`）— 缺檔保底（魔王/召喚物為數字 id，直接落到這層）。

每招的觸發類型決定泛型回退名（近戰系→swing、投射/區域/增益→cast、dash→dash、blink→blink、大招槽→ultimate）。
21 角色 × 4 技能 = 84 個簽名音，全部由 `tools/gen_sfx.py` 一次生成（見表後的對應）。

> 音色家族（擇要）：刀刃走「低基頻近諧波金屬＋強低頻 body」（重，不玻璃）；坦克盾擊／聖騎聖槌走
> **鈍器/大地**家族（低頻砰＋悶金屬，無刀刃鏜鏗）；弓/槍/法/聖光/召喚/星軌各有專屬合成。

## 檔案格式

- 載入順序：先找 `.mp3`（正式、相容性最佳），再找 `.wav`（佔位）；以能成功解碼的為主。
- 短音效請保持精簡（出手／命中通常 < 1 秒）；腳步請放「單一一步」的短音效（footstep1/2/3），程式會每步隨機選一 + 微調音高/音量。
- 目前 `.wav` 由 `tools/gen_sfx.py` 程序化合成（transient 起音／掃頻空氣聲／金屬泛音／殘響），非單純振盪器；可換成正式素材。

## 行為說明

- **空間化**：音效依與本地玩家的距離自動衰減音量，並做左右聲道定位（遠處角色較小聲）。
- **觸發分工**：出手／施法／大招／受傷／腳步由渲染端依角色狀態觸發；命中／死亡由特效事件觸發（不會重複發聲）。
- **音樂**：背景音樂在 `../music/`，與此處音效系統各自獨立。
